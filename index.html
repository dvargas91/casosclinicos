Bienvenido a la aplicación de análisis de casos clínicos

Esta aplicación guía, retroalimenta, monitoriza y evalúa su desempeño durante el análisis de un caso clínico.

Cada caso tiene varios módulos: motivo de consulta, enfermedad actual, antecedentes, revisión por sistemas; examen físico, diagnostico; pruebas diagnósticas, diagnostico final y gravedad de la enfermedad y tratamiento.

Antes de resolver el caso tenga en cuenta:

  • Cuando entra a un módulo debe revisar toda la información, contestar las preguntas y cerrar el grupo para poder avanzar al siguiente módulo.
  • Tiene un número de ingresos limitados, NO INGRESE SI NO DISPONE DE TIEMPO PARA TRABAJAR EN EL CASO.
  • Después de cerrar un módulo o cumplir el número de ingresos permitidos puede ver la información pero NO lo puede modificar.
  • Dispone de una página para tomar notas que estará disponible siempre que abra el caso. En ella podrá resumir la información que considere importante y explicar sus decisiones. Esto le servirá para tener el resumen del caso y para la revisión y discusión en clase o con el profesor.
  • El profesor le informará si el caso es evaluable o no, y si dispone o no de evaluación al finalizar el caso.

Para ingresar haga clic en login, ingrese su nombre de usuario y contraseña de la universidad.
